Mink.js

A Minimalistic DOM Library for Mobile Webkit.

Selectors

Mink wraps querySelectorAll and falls back to getElementById & getElementsByClassName when needed. Syntax is similar to jQuery.

    var foo = mink('#foo');
    var intro = mink('div > p.intro');

Methods

  • on
  • off
  • hide
  • show
  • append
  • remove
  • addClass
  • removeClass

Events

Registering to events are done by wrapping addEventListener, so all native DOM events are supported.

    mink('#foo').on('touchstart', function () {
      var foo = mink(this);
      foo.hide(); // will hide the element.
    });

Touch Events

Although touchstart, touchmove, and touchend are supported. It's probably best to use a library like Hammer.js which has abstracts gesture support.
